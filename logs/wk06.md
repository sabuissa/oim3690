# Week 6 Log

## What I built
- Built a to-do list app organized as two files: an HTML page (todo/index.html)
  and an external JavaScript file (todo/script.js). It can add tasks, check them
  off (strikethrough), and delete them.
- Set up the converter and discount pages for the JavaScript exercises.

## What I learned
- Why JavaScript is kept in a separate external file (script.js) and linked into
  the HTML, instead of written inline — it keeps the structure and the behavior
  separate and easier to read.
- How a page "reaches into" itself with document.getElementById to grab elements,
  then makes them react using addEventListener.

## AI interactions
- Used AI to build the to-do app, then read through the generated script.js so I
  could understand and explain how it adds, checks off, and deletes tasks.
