## Don't-know list (things from js-demo.html I didn't understand at first)

1. **addEventListener** — registers a function to run whenever a specified
   event (like a click or hover) happens on an element.

2. **`parseInt(mailbox.style.fontSize) || 30`** — extracts the numeric font
   size and falls back to 30 if none is set yet.

3. **Backticks with `${ }`** — let you embed variables directly inside a
   string instead of concatenating (joining) with `+`.
